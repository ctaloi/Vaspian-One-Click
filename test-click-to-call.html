Testing Instructions

  1. Install or reload the Vaspian One Click extension
  2. Phone numbers above should be highlighted and clickable
  3. Open the extension popup and go to Settings
  4. Click "Advanced Settings"
  5. Test the following scenarios:

Scenario 1: Global Disable

  • Uncheck "Enable Click-to-Call on Web Pages"
  • Reload this page
  • Phone numbers should no longer be clickable
  • Re-enable and reload to restore functionality

Scenario 2: Per-Site Disable

  • Ensure "Enable Click-to-Call on Web Pages" is checked
  • In the "Disabled Sites" section, enter the current hostname
  • For local file: enter "localhost" or the actual hostname
  • Click "Add"
  • Reload this page
  • Phone numbers should no longer be clickable
  • Remove the site from the disabled list and reload to restore

Expected Behavior

  • When enabled: Phone numbers are highlighted with an orange underline on hover
  • When enabled: Clicking a phone number shows a notification and initiates a call
  • When disabled globally: No phone number detection occurs
  • When disabled per-site: Phone numbers on that specific site are not clickable
  • Console message appears when click-to-call is disabled for a site
